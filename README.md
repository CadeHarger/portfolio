# Cade Harger — Personal Portfolio

My personal portfolio site, built with Vite + React + TypeScript and styled with Tailwind CSS v4.

Live at: **[placeholder-url.com](https://placeholder-url.com)**

## Stack

- **Vite + React + TypeScript** — SPA with client-side routing via `react-router-dom`
- **Tailwind CSS v4** — utility-first styling
- **HashRouter** — for GitHub Pages compatibility

## Running locally

```bash
npm install
npm run dev
```
